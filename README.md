## General info
Food Delivery Project is an example of RWD website created using ReactJs and SCSS - it presents website for Ordering the food including :

* Add to cart functionality
* Shopping cart functionality
* Product Filtering functionality
* Product page
* Cart and checkout page
* Contact page
* Login and Register page


## Technologies and tools used
Project is created with:

* ReactJs
* Redux Toolkit
* ReactStrap
* React-Paginate
* Remixicon
* Slick-Carousel
